# Pages Overview

The app has 5 feature tabs plus a "More" tab:

- **Today** — Dashboard with plant overview
- **Care Schedule** — Reminders and task management
- **Plant ID** — Camera-based plant identification
- **Species Library** — Searchable species reference
- **Plant Collection** — User's plants with add/edit
- **More** — Settings and About menu

# More

Demonstrates:

- A simple menu list
- Navigation to Settings and About sub-pages

The Settings screen also demonstrates:

- Theme selection
- Language selection (English/German/System)
- A persisted onboarding flag
- Navigation bar preferences (reorder, show/hide)
